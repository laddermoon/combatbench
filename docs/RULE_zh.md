# CombatBench 规则 V1.0（纯血条极简版）
 
## 一、任务目标
 
控制双足机器人，通过有效打击减少对方血量，先将对方血量降至 0 者获胜。
不判断倒地、不读秒、不设犯规、不干预姿态，一切由血量决定胜负。
 
## 二、胜负规则
 
1. **双方初始血量**：100 点
2. **获胜条件**：
   - 2.1 将对方血量先降至 0，则比赛立即结束，KO获胜。
   - 2.2 在比赛时间结束时，血量更高者获胜。
   - 2.3 时间结束血量相同：判定为平局。
3. **比赛时长**：每个回合时长 30 秒，进行 6 个回合。
4. **重置状态**：每个回合开始时，都会从初始位置开始（两个机器人距离两米远相向而立），不论上一局结束时是什么状态。
 
## 三、有效打击判定（唯一掉血逻辑）
 
### 1. 允许的攻击部位（打击方）
 
仅以下部位发动的攻击可造成伤害：
- 手（hands）
- 前臂（forearms）
- 肘部（elbows）
- 上臂（upper arms）
- 脚（feet）
- 小腿（shins）
- 膝盖（knees）
- 大腿（thighs）
 
**注意**：躯干、头部均不能作为有效攻击部位。使用躯干和头部击中对方，对方不掉血。
 
### 2. 有效受击部位（被打击方）
 
仅以下部位被击中会掉血：
- 头部（head）
- 躯干（torso，含胸、腹、腰、背）
 
其它部位被击中一律不掉血。
 
### 3. 物理条件（真打击判定）
 
必须同时满足以下条件：
- **高速瞬间碰撞**：碰撞相对速度必须大于设定阈值（排除慢碰、挤压）。
- **非持续接触**：一次碰撞事件只结算 1 次伤害，持续贴合不重复掉血。
 
### 4. 伤害数值
 
- **头部被击中**：-3 点
- **躯干被击中**：-1 点
 
*(在V1.0简化版中，伤害由有效碰撞直接触发，不区分轻击或重击。)*
 
## 四、姿态与行为规则（全部不限制）
 
以下行为均不判负、不扣分、不惩罚、不重置：
- 倒地、翻滚、地面进攻、地面防守。
- 缠抱、贴近、压制、拉拽。
- 用头撞击、用躯干顶撞（不掉血，但允许发生）。
- 任何姿态、任何移动方式。
 
AI 可自由演化最优策略，不需要模仿人类格斗。
 
## 五、物理与执行规则
 
1. **物理步长**：固定 500Hz，保证物理一致性。
2. **策略决策频率**：20Hz（每 50 毫秒一步）。
3. **超时行为**：超期未输出动作则自动保持上一动作。
4. **公平性**：全局物理参数、模型参数对双方完全一致，保证绝对公平。

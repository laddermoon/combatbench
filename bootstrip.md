# combatbench

这个项目是仿真人形机器人对战的Benchmark平台中的开源部分。
这部分开源的主要包括
1. 仿真环境的实现，
2. 仿真环境的使用指南，
3. 比赛的规则说明，
4. 用户提交策略的说明，
5. 以及提交工具的分发，
6. 还有到平台Web页面的引导。

项目规范：
1. 做成一个标准的开源项目（符合大家对于开源项目组织形态的理解）
2. 主要资源来自于：things/combatbench/simulator_humanoid21 
   把里面有用的代码做规范化之后整理到这个项目里。并且保持原来的功能可用。
3. 要能应对未来的可扩展性，未来的扩展方向是会添加 宇树G1的支持， 然后会添加纯视觉观测的支持（关于对手的信息，只有主观摄像头的视觉信息）。
